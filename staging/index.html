    UX Experiments

    I like to think about User Experience Design as a language that enables communication between Humans and any Information system. These experiments are side projects I was involved in for fun and to explore the different facets of User Experience and Interaction Design across various different platforms.


    Multi-touch Table Interaction

    Extending physical gameplay to digital world using tracked physical tokens on a interactive touch table. This allows us to share characteristics of the physical tokens to the virtual space, thereby enhancing the experience.

    • See Demo

    Natural VR gameplay

    An exploration into creating interventions for natural user experience in Virtual Reality gameplay. For the prototype, body tracking using Kinect was used to detect interactions with the server being node.js

    • See Demo

    Tangible Interactions

    Interchangable instrument modules provides the affordance required for digital music to be created on a smartphone. A combination of acoustic and motion tracking is employed for the prototype
