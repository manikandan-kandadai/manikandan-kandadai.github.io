Rakshaa

Interaction Design, Visual Design

An application that everyone needs yet no one wants to have on their phones. 

This is true with most of the emergency/safety apps currently present in the app store. The experience of using the app supersedes any utility present in an app. This is my attempt to change the way women perceive and use emergency app.

This project is a semester long individual project and is a result of recruitment, interviews, user testing, iteration informed by user and peer feedbacks.
